Правила работы с расширением

  • Выделите нужные строки, где указаны БИНы для которых нужно обновить данные. При большом количестве запросов, источник данных может временно заблокировать отправку.
  • Нажмите кнопку "Получить данные"
  • Столбцы A-F используются для текущей информации, новые столбцы будут добавлены справа. Первая строка используется для заголовков. Изменяя эту структуру повышается риск неисправной работы приложения. При необходимости модификаций может потребоваться изменение кода.
  • Когда нужные данные получены и таблица готова для анализа, нажимите кнопку "Создать виды для текущего листа", чтобы отобразить текущий лист на новые листы с отдельной сортировкой столбцов..
