Правила работы с расширением

  • Выделите нужные строки, где указаны БИНы для которых нужно обновить данные. Не выбирайте слишком много за раз, т.к. это займет время, кроме того, при большом количестве запросов, источник данных может временно заблокировать отправку данных.
  • Нажмите кнопку "Получить данные"
  • Столбцы A-F используются для текущей информации, новые столбцы будут добавлены справа. Первая строка используется для заголовков. Изменяя эту структуру повышается риск неисправной работы приложения. При необходимости модификаций может потребоваться изменение кода.
